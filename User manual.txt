# User Manual for Word Search Puzzle Game

## Introduction
The **Word Search Puzzle Game** is a simple and engaging program where players search for hidden words in a 2D grid of letters. This manual provides an overview of how to play the game and save your progress.

## How to Play

### Objective
Find all the hidden words in the grid. Words can appear in any direction: horizontal, vertical, or diagonal.

### Steps to Play
1. Start the game by running the compiled program:
2. View the grid of letters and the list of words to find.
3. Enter a word to search for in the grid.
4. The program will check the grid and mark the word if found.
5. Continue searching until all words are found or you decide to stop.
6. Save your progress automatically when the game ends.

## Features
- Words can be searched in all directions.
- Found words are marked in the grid.
- Progress is saved to a file named `word_search_results.txt`.

## Saving Results
When you finish, the program generates a file containing:
- The puzzle grid.
- A list of words you found.

## Troubleshooting
- Ensure your C++ compiler is installed and properly configured.
- Make sure the binary file has execute permissions.
